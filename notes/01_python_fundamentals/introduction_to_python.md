# 🎯 What is Python?

Python is a **high-level, interpreted programming language** designed to be **simple, readable, and powerful**.

It allows programmers to write **clear and logical code**.

Example Python code:

```python
print("Hello, World!")
```

Output:

```
Hello, World!
```

Just **one line of code** to print text 🎉.

---

# 🧠 Who Created Python?

Python was created by **Guido van Rossum** in **1991**.

His goal was to build a language that is:

✔ Easy to read  
✔ Easy to learn  
✔ Powerful for large projects  

Fun fact 🤓

The name **Python** comes from the comedy show:

```
Monty Python’s Flying Circus
```

Not from the snake 🐍.

## 2️⃣ Interpreted Language
