# 🧠 Before Installing Python

Check if Python is **already installed** on your system.

### Step 1

Open **Command Prompt**.

Press:

```
Windows + R
```

Type:

```
cmd
```

---

### Step 2

Run this command:

```bash
python --version
```

or

```bash
python -V
```

Example output:

```
Python 3.12.2
```

If you see a version → Python is already installed ✅

If not → install Python.

---

# 💻 Installing Python on Windows

Follow these steps carefully.

---

# Step 1 — Go to Official Website

Open:

```
https://www.python.org/downloads/
```

You will see a big button:

```
Download Python
```

Click it.

---

# Step 2 — Download Installer

Example file:

```
python-3.12.x-amd64.exe
```

Download completes 📥.

---

# Step 3 — Run Installer

Double click the downloaded file.

You will see the installer window.

⚠️ **VERY IMPORTANT STEP**

Check this box:

```
Add Python to PATH
```

Then click:

```
Install Now
```

---

# Step 4 — Installation Process

Python will install automatically.

You will see:

```
Setup was successful
```

Click **Close**.

Python is now installed 🎉.

---

# 🧪 Verify Python Installation

Open **Command Prompt** again.

Run:

```bash
python --version
```

Example output:

```
Python 3.12.2
```

Success ✅

---

# 🐍 Running Your First Python Program

Run Python interactive shell:

```bash
python
```

You will see:

```
Python 3.x.x
>>>
```

Now type:

```python
print("Hello Python")
```

Output:

```
Hello Python
```

Congratulations 🎉  
You ran your first Python program.

---

# ⚡ Python Interactive Mode

Python shell allows testing code quickly.

Example:

```python
>>> 5 + 5
10
```

Example:

```python
>>> name = "Tarra"
>>> print(name)
Tarra
```

Exit shell:

```python
exit()
```

or press:

```
Ctrl + Z + Enter
```

---

# 📂 Running Python File

# ⚠️ Common Installation Problems

### Problem 1

```
python is not recognized
```

Solution:

You forgot to check:

```
Add Python to PATH
```

Reinstall Python.

---

### Problem 2

Multiple Python versions.

Check using:

```bash
where python
```
